# Rumaliwala

Premium Rumali & Islamic Accessories - catalogue with WhatsApp ordering.

**Live site:** https://hii2023.github.io/rumaliwala/
**Admin panel:** https://hii2023.github.io/rumaliwala/#admin

## Managing products
Open the admin page, enter the store passcode, then add, edit, delete or hide products.
Changes go live for all customers instantly (backed by Supabase).

## Before sharing widely
- Set the real WhatsApp / call number in the `CONFIG` block of `index.html`.
- Add product photos: open a product in the admin and paste an Image URL.

Single-file static site (`index.html`). No build step.
